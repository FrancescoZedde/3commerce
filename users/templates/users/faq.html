  • 1. Log in to your website.
  • 2. Go to WooCommerce > Settings > Advanced > REST API.
  • 3. Click on the "Add Key" button.
  • 4. Enter a description for your API key (e.g. "My API Key").
  • 5. Select the user that you want to associate with this API key.
  • 6. Choose the permissions that you want to grant to this API key (read, write, or read/write).
  • 7. Click on the "Generate API Key" button.
  • 8. Your new API key will be displayed on the screen. Make sure to copy it and save it in a safe place, as you won't be able to see it again later.
  • 9. Copy and paste the public key and private key into the Account page. Click "Connect," if the connection was successful you will receive a confirmation message.
